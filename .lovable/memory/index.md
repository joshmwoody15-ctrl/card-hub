# Card Hub — Project Memory

## Core
App name: Card Hub. Sports card sales tracker across 8 marketplaces (eBay, Fanatics Collect, Beckett, COMC, Whatnot, Sportlots, MySlabs, CollX).
Palette: reds (primary crimson), golds (accent), black, greys, white. Tokens in src/styles.css; never hardcode colors.
Typography: Bebas Neue display, Inter body, JetBrains Mono for numbers (.tabular / font-mono).
Design direction: "Museum-grade clinical" — light, data-dense, professional. Defined in oklch.
Verified peer-to-peer cash deals use --verified (emerald) accent and ShieldCheck icon.
Data source: Firecrawl (search per marketplace). Lovable Cloud for auth + Postgres.
